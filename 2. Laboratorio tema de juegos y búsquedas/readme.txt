IA para jugar tres en raya(tic tac toe) usando el algoritmo minmax
Codigo en C++ en ZinjaI usando la libreria glut para OpenGL
-------
Comando recomendado para linux: g++ main.cpp -o firstOpenGlApp -lglut -lGLU -lGL
-------
para jugar debera elegir la dificultad (facil, medio o dificil), luego elegir si quieres jugar primero o dejar a la maquina jugar primero
y en la interfaz hacer click en la casilla que desea jugar y la computadora jugara por su cuenta, enjoy
Comandos:
tecla "esc" para salir del programa
tecla "r" para reiniciar el juego
